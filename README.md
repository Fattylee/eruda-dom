# eruda-dom

Eruda plugin for navigating dom tree.

## Demo

Browse it on your phone: 
[http://eruda.liriliri.io/](http://eruda.liriliri.io/)

## Install

```bash
npm install eruda-dom --save
```

```javascript
eruda.add(erudaDom);
```

Make sure Eruda is loaded before this plugin, otherwise won't work.
